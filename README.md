Chatty App
=====================

A clone of Slack.

### Usage

Clone the repository.

```
git clone git@github.com:kwan14/chattyApp.git

```

Install the dependencies and start the server.

```
cd chattyApp
npm install
npm start
cd ./chatty_server
npm install
node server.js
open http://localhost:3000

```
